## Iniciando
Certifique-se de estar na pasta raíz do projeto

- Instale as dependências:

``` npm install ```

- Modifique as variáveis ambiente:

``` cp .env.example .env ```

``` nano .env ```

- Iniciando
``` npm run start ```

## Testes

- Inicia todos os testes:

``` npm run test ``` 

- Inicia testes de integração:

``` npm run test:integration ```

- Inicia testes unitários:

``` npm run test:unit ```
